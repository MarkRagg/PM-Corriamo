# PM-Corriamo

Progetto di Project Management che riguarda la simulazione dello sviluppo di una web app per la gestione degli atleti in un poliambulatorio sportivo.
La simulazione riguarda tutte le fasi dello sviluppo, a partire dallo scoping, fino ad arrivare alla formale chiusura del progetto.
